jquery基本知识

发表于 2018-01-10 | 分类于 jquery

这是jquery的基本知识，记录下来为了加深印象，有些不全，后续一些方法，持续更新

jquery对象

DOM对象是文档对象模型，将HTML解析成DOM树，通过document.getElementById()等方式获取到HTML元素节点；而jquery对象则是jquery包装DOM对象后产生的对象，是jquery独有的。

注意

jquery对象中无法使用DOM对象的任何方法；DOM对象同样如此

选择器

jQuery选择器完全继承css的风格，同时支持css1-css3，兼容各种浏览器，十分快捷的找到特定的DOM元素，添加行为，jquery选择器分为4中，下面依次介绍。

基本选择器

基本选择器包含css常见的，id，class，元素标签，通配符选择器

层次选择器

通过DOM之间的层次关系来选择获取特定元素，像后代元素，子元素，相邻元素，同辈元素

过滤选择器

过滤则是用来筛选的，和伪类选择器语法相同，以: 开头

基本过滤

内容过滤

规则体现在其包含的子元素或文本内容上

可见性过滤

根据元素的可见性来过滤

属性过滤选择器

根据元素的属性来过滤

子元素过滤器

对子元素的过滤选择

表单对象属性过滤器

对所选择的表单进行过滤

表单选择器

灵活调用表单的元素

注意事项

  • 选择器中含有· # ( ) ]等特殊字符用\\转义符进行转义
  • 后代选择器和过滤选择器的空格

DOM操作

DOM即为文档对象模型，通过DOM操作节点达到相关的目的。一般DOM操作分为三种，DOM Core 即通过document.getElementById等相关操作；HTML_DOM 即通过HTML document.form[0]等等； CSS_DOM 即设置css相关样式，div.style.color=”red”等等。

查找节点

查找元素节点 直接 $(“ul li:eq(1)”)

查找属性节点 直接attr()方法 当attr里的参数为1个即为查找 2个即为设置 $(“div”).attr(“id”)

创建节点

插入节点

append() 在节点内部之后添加节点 appendTo 元素位置倒置

prepend() 在节点内部之前添加节点 prependTo 元素位置倒置

after() 在节点之后添加节点 insertAfter 元素位置倒置

before() 在节点之前添加节点 insertBefore 元素位置倒置

删除节点

remove() $(“ul li:eq(1)”).remove() 直接删除整个节点包含子节点，返回的是这个节点，可以引用的；同时，remove里可以传递参数，来进行选择性的删除

detach() 通过删除整个节点，但是节点绑定的事件和一些数据依旧是保存的

empty() 清空节点，清空节点中所有后代节点

复制节点

clone 有参数，当clone(true)时候则复制的节点依旧有原来的事件

替换节点

replaceWith() 将节点替换成想要的节点

replaceAll() 颠倒上述操作

包裹节点

wrap 将所有元素进行单独的包裹 $(“div”).wrap(“//“) 用b标签将div包裹住

wrapAll 将所有元素进行整体的包裹

wrapInner 让其成为最里面的一层，分别的反包裹

获取设置删除属性

获取节点时说过了

removeAttr()即为删除属性

获取设置追加样式

attr 直接性的获取设置样式 $(“div”).attr(“class”,value)

addClass追加样式，直接保持两个样式 如果设置相同属性则后者覆盖前者

删除切换判断样式

removeClass 直接删除样式 $(“div”).removeClass(“big more”) 不带参数则class直接都被删除

toggleClass 切换样式，每次点击就会换个样式

hasClass 判断样式，返回的是布尔值 当然可以用is $(“div”).hasClass(“high”) 如果用is则是”.high”

设置和获取HTML和文本的值

html 类似js中的innerHTML 有参数则是设置

text 直接获取文本内容 有参数则是设置

val 直接获取value值 有参数则是设置 当然val还可以对表单控件 checkbox select radio 相应的选项进行选择

遍历节点

children() 取得元素的子元素集合 不考虑后代元素

next() 相邻的下一个同辈元素

prev() 相邻的上一个同辈元素

siblings 相邻的所有同辈元素

parent 获得每个匹配元素的父级元素

parents 获得每个匹配元素的祖先元素，返回多个parent

closest 从元素本身开始逐级向上匹配，返回最先匹配的祖先元素

CSS方法读取设置样式

直接设置即可 $(“div”).css({“background:red,fontSize:30px”})

offset方法 对于当前视窗的相对偏移量 包含top和left两个属性 只对可见元素有效

position方法 对于有relative或absolute的父级相对偏移量 包含top和left两个属性

scrollTop和scrollLeft 分别获取元素滚动条距顶端和左侧的距离 对其指定参数到达一定的距离处

事件和动画

事件绑定

事件绑定正式为bind()方法，有3个参数，第一为事件类型，第二为可选参数，作为Event.data属性值传递给事件对象的额外数据对象，第三为执行的函数。

特点

this 和js中的this是一样的，指当前执行的DOM元素

事件可以链式操作

某个元素多次使用，定义成局部变量

合成事件

hover方法 用于模拟光标悬停事件，两个函数参数

toggle方法 用于模拟连续单击事件，多个函数参数；另一个作用是切换元素的可见状态(类似show和hidden),可见状态时可以传递数字代表动画

事件冒泡

不支持事件捕获，直接冒泡

stopPropagation方法 阻止事件冒泡

preventDefault方法 阻止默认行为

return false 直接设置两者

事件对象的属性

event.type 获取事件的类型

event.preventDefault 阻止默认行为

event.stopPropagation 阻止事件冒泡

event.target 获取触发事件的元素

event.relatedTarget 获取相关元素

event.pageX 和 event.pageY 相对于页面的X坐标和y坐标，加上滚动条

event.which 获取鼠标的左、中、右键

event.metaKey 封装的获取Ctrl键

移除事件

直接unbind() 第一个参数为事件类型，第二个参数是移除的函数，没有则移除全部的绑定事件

当然一次性的绑定事件直接用one即可

模拟操作

通过模拟用户来操作，直接使用trigger方法

可以出发自定义事件；可以传递数据；可以执行默认操作

绑定多个事件类型

事件用空格分开即可；添加命名空间；

jquery 动画

show和hide 改变元素的display属性

三个固定参数为slow normal fast 为600 400 200

fadeIn和fadeOut 改变元素的不透明度

sideUp和sideDown 改变元素的高度 自上而下和自下而上

当然，还有其他方法

slideToggle fadeTo(600,0.3) fadeToggle

自定义动画

直接animate，包含3个参数，params speed callback

回调的话，直接加上callback，在里面执行其他的方法，直接在后面加则是立即执行。

停止动画

stop方法，两个参数，clearQueue和gotoEnd，都是布尔值；主要是结束当前正在进行的动画，并立即执行队列中的下一个动画。clearQueue直接清除未执行的动画，gotoEnd直接将动画到最后状态。

判断动画状态

判断元素是否在动画状态，不在动画状态才添加动画

延迟动画

delay()进行动画延迟操作

持续更新的知识点

bind live delegate on 的区别

bind

单事件处理通常简写，直接事件；多事件处理，可以用空格，链式操作，还有大括号json的表示方式。

缺点：绑定到所有选出的元素；不会绑定到后来动态添加的元素；效率问题

live

jquery不推荐，高版本不支持，事件代理，适合列表

delegate

更小范围的事件代理，不会将event全部绑定在document上，可以自由选择，动态添加的元素依然享有事件机制

on

推荐使用这个方法，动态添加的元素依然享有事件机制，不过要放在里面

mouseenter和mouseover的区别

区别 mouseenter不冒泡，mouseover冒泡，当元素有子元素时，前者不会触发，而后者会触发

模拟 由于mouseenter的兼容性，需要模仿mouseenter,利用relatedTarget的值，当其值不是目标元素，也不是目标元素的子元素，说明已经移入目标元素而不是元素内部移动 。原生js的contains方法

contains ele.contains表示传入的节点是否为该节点的后代节点,返回布尔值

一些其他方法

error() 当元素发生错误(没有正确载入)，发生error事件

串联操作

end() 返回上一次jquery对象

add() 增加对象到当前jquery对象，相当于并列的选择,分割

contents() 查找子节点元素，返回的是文本节点及所生成的jquery对象中的页面元素，返回的是对象

andSelf() 新增自身对象到当前的jquery对象

filter() 将匹配元素集合缩减为匹配指定选择器的元素，和not()方法相对的

map() 指定函数处理数组的每个元素，并将处理结果封装为新的数组返回
