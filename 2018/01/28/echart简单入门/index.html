echart简单入门

Jan 28 2018

echart，百度出品的可视化图表库，基于html5 canvas的纯javascript的作品

各个元素的配置

两个方法

通过echart.init()方法直接获取容器

通过echart.setOption()方法直接生成配置项

title 标题组件

分为主标题text和副标题subtext

超链接直接link/sublink target/subtarget

内边距padding 默认为5 接受数组设定上右下左边距

主副标题间距 itemGap 组件所有图形的z值 z

grid组件的方位距离 left、right、top、bottom

标题背景，边框颜色 backgroundColor borderColor borderWidth borderRadius

图形阴影的模糊大小 shadowBlur shadowOffsetX shadowOffsetY

文本设置 textStyle/subtextStyle 很多调整属性fontSize fontWeight fontFamily等等

legend 图例组件

图例类型 type 分为plain/scroll

图例方位距离 left、right、top、bottom

图例长宽 width height

图例列表的朝向和对齐 orient分为horizontal/vertical align

图例间隔 itemGap

图例的数据数组 data[i]

grid 直角坐标系

xAxis x轴 YAxis y轴

x轴所在的grid的索引 gridIndex

x轴的位置 position 可选top、bottom

坐标轴类型 type 可选 value数值轴、category类目轴和data设置、time时间轴、log对数轴

坐标轴名称 name nameLocation：start、middle、end 当然这里还有很多，见官方文档

坐标轴线相关设置 axisLine 坐标轴刻度相关设置axisTick

坐标轴刻度标签axisLabel

类目数据data针对category

polar 极坐标 用于散点图和折线图

极坐标的中心坐标 cneter[400,300]

极坐标的半径 radius[内半径，外半径]

radiusAxis angleAxis

和XAxis的属性差不多

radar 用于雷达图

每个轴都是一个单独的维度，可以通过name axisLine axisTick axisLabel splitLine splitArea来配置

第一个指示器轴的角度 startAngle

雷达图的绘制 shape 分为 polygon/circle

雷达图的指示器 indicator 指定多个维度

tooltip 提示框组件

设置在全局即tooltip

设置在坐标系即grid.tooltip polar.tooltip single.tooltip series.tooltip series.data.tooltip

是否显示 show

触发类型 trigger 分为item/axis

触发条件 triggerOn 分为Mousemove/click/mousemove|click/none

geo 地理坐标组件

支持散点图，线集

parallel 平行坐标系

用于高维数据

toolbox 工具栏

导出图片，数据视图，动态类型切换、数据区域缩放，重置

show orient itemSize itemGap showTitle

各工具配置项 feature

saveAsImage restore dataView dataZoom magicType(type:line/bar/stack/tiled) brush

补充brush

type 有 rect polygon lineX lineY keep clear

brush 区域选择组件

选择图中一部分数据，向用户展示被选中数据或计算结果

按钮 toolbox brushLink 不同系间的联动

series 系列列表

通过设置type即可决定图表类型

line 折线 bar 柱状 pie 饼图 scatter 散点图 effectScatter 带有动画的散点图

radar 雷达图 tree 树状图 sunburst 多层的环形图 boxplot 箱型图 candlestick k线图

hotmap 热力图 map 地理区域数据 parallel 平行坐标系 lines 航线线路可视化 graph 节点间的关系

funnel 漏斗图 gauge 仪表图 pictorialBar 象形柱图 themeRiver 主题河流 custom 自定义系列
