echart的进阶

图表分为很多类型，在echart有许多，比如：line、bar、pie、scatter、map、candlestick、radar、boxplot、heartmap、tree、funnel等等，本文以最常见的几种图表类型为例，分析图表的一些属性。

line

折线图表

boundaryGap 坐标轴两边留白，类目轴：true/false 非类目轴：两个值的数组，表示最小值和最大值的范围 [‘20%’,’30%’]

areaStyle 区域填充样式，自定义color origin shadowBlur shadowColor opacity 等等

stack 数据堆叠，同类目轴上系列配置相同的stack值后，后一系列值再前一个系列值上相加 默认为null

smooth 折线是否为曲线形式表现 默认为false

补充：通用的缩放属性

dataZoom

type 为 inside 时为内置型数据区域缩放

type为 slider 时为缓动型数据区域缩放
