各种各样的AE表达式，轻松解决

ae表达式

ee显示表达式

常用表达式

value 属性数值的灵活控制 value [0/1/2……]

layer index 用于图层之间的表达式关联

使用ctrl+d将该图层复制多个，因为新复制的图层序号默认为排在首位，所以新出现的图层始终以300的间距向右排列。

time 通过获取时间轴时间变动，将动画赋予属性值；比如time用于旋转则每秒n角度，还可以运用到不透明度，位置，缩放等等

valueAtTime 对动画播放时间实现精准控制

sourceRectAtTime 通过获取图层内容块宽高的即时变化，将数值赋予其他属性

loopin loopout 关键帧动画循环

一般情况下直接loopOut从开始帧到结束帧的循环；

linear ease 动画插值转换计算方法

这个表达式无法直接使用，需要定义相应的值。

t：需要绑定的对象的属性值，直接可以使用拉索工具链接到控制按钮的Y值属性上。

tmin&tmax：定义属性大小值范围，也就是按钮Y值在滑杆两个端点位置Y的值。

value1&value2：定义转换属性的变动值，也就是本案例中该圆旋转的角度范围。

wiggle 随机晃动摇摆 wiggle(频率，幅度)

表达式控制

滑块控制 可以结合3d图层控制住几个直接的间隔距离

点控制

角度控制

图层控制 了解几个图层间的位置等其他属性
